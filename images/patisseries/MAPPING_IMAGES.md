## Images par défaut (sections générales)

- `patisserie-defaut.jpg` - Image utilisée pour :
  - Hero section (index.html et recettes.html)
  - Section histoire
  - Section engagement
  - Section B2B
  - Section commandes spéciales
  - Section livraison
  - Fallback pour toutes les images manquantes

## Instructions

1. Placez toutes vos images dans le dossier `images/patisseries/`
2. Nommez-les exactement comme indiqué ci-dessus (respectez la casse et les accents)
3. Format recommandé : JPG ou PNG, 1200x800px minimum, optimisé pour le web (< 500KB)
4. Si une image spécifique n'est pas trouvée, le site utilisera automatiquement `patisserie-defaut.jpg`
