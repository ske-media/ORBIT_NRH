# Images de pâtisseries

Ce dossier contient les images des pâtisseries pour le site Chef Bulier.

## Structure recommandée

Placez vos images dans ce dossier avec les noms suivants :

- `pain-sans-gluten.jpg` - Pain sans gluten
- `brioche-ig-bas.jpg` - Brioche IG bas
- `macarons-assortis.jpg` - Macarons assortis
- `eclair-chocolat.jpg` - Éclair au chocolat
- `cookies-vegan.jpg` - Cookies vegan sans gluten

Et pour toutes les autres recettes de la page recettes.html.

## Format recommandé

- Format : JPG ou PNG
- Taille : 1200x800px minimum
- Poids : Optimisé pour le web (< 500KB)
- Qualité : Haute résolution pour un rendu premium

## Fallback

Si une image n'est pas trouvée, le site utilisera automatiquement l'image par défaut (patisserie-defaut.jpg).
